## Features

* Shows a notification with AirPods status when they are connected
* Respects your privacy
* Free and Open Source software
* Supports Dark Theme

## Supported devices

* Apple AirPods 1st gen
* Apple AirPods 2nd gen
* Apple AirPods 3rd gen
* Apple AirPods Pro
* Apple AirPods Pro 2
* Apple AirPods Pro 3
* Apple AirPods Max
* Beats X
* Beats Flex
* Beats Solo 3
* Beats Studio 3
* Powerbeats 3
* Powerbeats Pro
